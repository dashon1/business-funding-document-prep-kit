# SECTION 2: BANK STATEMENT PREP SHEET

## 📊 Preparing Your Bank Statements for Lender Review

---

### Why Bank Statements Matter

Lenders analyze your bank statements to verify:

- **Cash Flow Consistency** — Regular deposits and stable balances
- ** Business Revenue Verification** — Income matches reported figures
- **Expense Management** — Reasonable and appropriate spending
- **Cash Reserve Maintenance** — Ability to cover obligations
- **Payment History** — How you handle financial relationships

---

## ✅ Pre-Submission Checklist

### Before Submitting, Ensure:

| Task | Complete |
|------|----------|
| Request all pages (front and back) of statements | ☐ |
| Verify all transactions are legible | ☐ |
| Remove or redact any personal transactions you prefer not to share | ☐ |
| Ensure business name matches exactly on all statements | ☐ |
| Verify dates cover the required period (typically 6-12 months) | ☐ |
| Calculate average monthly balances for each statement | ☐ |
| Note any large deposits or withdrawals | ☐ |
| Identify any returned deposits or overdrafts | ☐ |

## 🔍 Red Flags to Address Before Submission

### Things Lenders May Question:

| Issue | Action to Take |
|-------|----------------|
| Large Unexplained Deposits | Provide documentation or explanation letter |
| Significant Balance Fluctuations | Prepare narrative explaining irregular pattern |
| NSF/Overdraft Fees | Document as one-time occurrence if applicable |
| Pattern of Low Balances | Show business seasonality or planning |
| Large Cash Withdrawals | Prepare receipts and business justification |
| Irregular Deposit Pattern | Explain industry payment cycles |

# SECTION 7: FOLDER ORGANIZATION GUIDE

## 📁 Physical and Digital Document Organization System

---

### Why Organization Matters

**Professional Appearance** — Lenders notice when businesses are disorganized

**Faster Processing** — Complete packages get reviewed faster

**Confidence Building** — Organized documentation demonstrates professionalism

**Problem Prevention** — Reduces errors and missing information

---

## 🗂️ PHYSICAL FILE SYSTEM

### Create these main folders:

| Folder | Contents | Tab Color |
|--------|----------|-----------|
| **1. Business Formation** | Articles, EIN, licenses, agreements | ☐ Blue |
| **2. Financial Statements** | Tax returns, P&L, balance sheets | ☐ Green |
| **3. Bank Statements** | All accounts, all months | ☐ Yellow |
| **4. Debt Documents** | Loans, credit cards, leases | ☐ Red |
| **5. Insurance** | Policies, claims history | ☐ Orange |
| **6. Contracts** | Vendor, customer, lease agreements | ☐ Purple |
| **7. Tax Records** | All filed returns, correspondence | ☐ Brown |
| **8. Licenses & Permits** | All business licenses | ☐ Gray |

---

### Inside Each Folder, Create Sub-Folders:

**Example Structure:**

```
├── Financial Statements (Main Folder)
│   ├── 2023 Tax Returns
│   ├── 2022 Tax Returns
│   ├── Monthly P&L Statements
│   │   ├── January 2023
│   │   ├── February 2023
│   │   └── ...
│   ├── Balance Sheets
│   └── Financial Projections
```

---

## 💻 DIGITAL FILE SYSTEM

### Recommended Folder Structure:

| Main Folder | Sub-Folders | Files |
|-------------|-------------|-------|
| **Business_Funding** | | Master folder |
| ├── **01_Formation** | | Articles, EIN, licenses |
| ├── **02_Financials** | Annual, Monthly, Projections | Statements, returns |
| ├── **03_Banking** | Statements, Correspondence | Bank records |
| ├── **04_Debt** | Active, Paid-Off, Applications | Loan documents |
| ├── **05_Insurance** | Current, Historical | Policy documents |
| ├── **06_Contracts** | Vendors, Customers, Leases | All agreements |
| ├── **07_Tax** | Federal, State, Annual filings | Tax records |
| └── **08_Licenses** | Federal, State, Local | Permits, licenses |

---

### File Naming Convention

**Use consistent naming:** `YYMMDD_DocumentType_BusinessName.ext`

**Examples:**
- `230101_TaxReturn_ABCBusiness.pdf`
- `230315_BankStatement_ABCBusiness.pdf`
- `230520_LoanApp_ABCBusiness.pdf`
