# Folder Organization Guide

**Set Up a System That Makes You Look Professional—and Saves You Hours Every Time You Need Documents**

---

## Why Organization Matters

When you apply for funding, lenders don't just evaluate your business—they evaluate you. Business owners who can produce documents quickly and accurately demonstrate operational competence. Those who scramble for hours looking for statements appear disorganized and potentially risky.

This guide teaches you how to organize your financial documents so you're always ready. A well-organized system takes one weekend to set up and saves countless hours every time you need to pull documents together.

---

## The Basic Framework

Your document organization system has two components working in parallel:

1. **Physical Files** — Paper documents, printed statements, contracts, and official paperwork
2. **Digital Files** — Electronic copies stored on your computer or cloud storage

Both systems follow the same folder structure, making it easy to find documents whether you're looking at a physical file cabinet or a computer folder.

**Key Principle:** Consistency is everything. Use the same folder names, same structure, and same naming conventions everywhere. When everything has a predictable place, you'll never waste time searching.

---

## The Master Folder Structure

Create one master folder on your computer named "Business Funding Documents"

Inside that folder, create the following subfolders:

```
Business Funding Documents/
├── 00_Reference_Materials/
├── 01_Business_Identity/
├── 02_Financial_Statements/
├── 03_Bank_Documents/
├── 04_Debt_Documentation/
├── 05_Revenue_Records/
├── 06_Collateral_Documentation/
├── 07_Personal_Financials/
├── 08_Business_Plan/
├── 09_Tax_Returns/
├── 10_Insurance/
├── 11_Licenses_Permits/
└── 12_Submission_Ready/
```

For physical files, create the same folder structure in a filing cabinet or binder system. Label each folder clearly.

## Folder-by-Folder Breakdown

### Folder 00: Reference Materials

**Purpose:** Store guides, instructions, and reference documents that help you maintain your system

**Contents:**

- This organization guide (keep a copy here)
- Funding application checklist (your master reference)
- Lender requirements cheat sheet (notes on what different lenders need)
- Contact directory (banker names, accountant info, attorney info)

**How to Use:** When preparing for a funding application, reference these materials first to remind yourself of the complete process.

## Naming Conventions Master Reference

**Date Format:** Always use YYYY_MM_DD format for consistency
**Example:** 2025_03_15

**Key Elements:**

- Document type (what it is)
- Context (who, where, which)
- Date (when)
- Version (if applicable)

**Never use:** Spaces in file names, vague names like "Doc1.pdf" or "Statement.pdf"

## System Maintenance Schedule

### Weekly Tasks (5 minutes)

- Move new digital documents into proper folders
- File physical documents immediately when received
- Update any incomplete file naming

### Monthly Tasks (15 minutes)

- Review 03 Bank Documents folder
- Move last month to "Archive" subfolder
- Add any new statements to 02 Financial Statements

### Quarterly Tasks (30 minutes)

- Update financial statement folders
- Verify all documents are current
- Review folder structure for needed additions

### Annual Tasks (1 hour)

- Archive old documents (move to yearly archive folders)
- Clean up and delete unnecessary files
- Review and update folder structure if needed
- Update tax return folders with new year

## Quick Access: Documents for Common Requests

When lenders ask for common document packages, here's how to quickly assemble them:

**For Standard Business Loan (most common):**

1. Go to 01 Business Identity — grab incorporation docs
2. Go to 09 Tax Returns — grab last 3 years returns
3. Go to 02 Financial Statements — grab monthly P&Ls for 12 months and current balance sheet
4. Go to 03 Bank Documents — grab last 6 months statements
5. Go to 04 Debt Documentation — grab all debt statements
6. Go to 08 Business Plan — grab executive summary and projections
7. Go to 12 Submission Ready — create lender-specific folder

**For SBA Loans:**
Add: 07 Personal Financials (personal returns, personal statements, personal balance sheet)

**For Equipment Financing:**
Add: 06 Collateral Documentation (equipment info)

## Cloud Storage Options

If you use cloud storage (Dropbox, Google Drive, OneDrive, iCloud), create the same folder structure in your cloud drive.

**Sync Strategy:**

- Keep 12_Submission_Ready folder synced across devices
- Make sure financial folders sync automatically
- Keep personal folder local or on private cloud

**Security Consideration:**
Financial documents contain sensitive data. Use cloud storage with strong passwords and two-factor authentication. Consider encrypting especially sensitive folders.

## Emergency Access Protocol

If you needed to pull together all funding documents today, how quickly could you do it?

With this system: 15-30 minutes
Without this system: Several hours to days

**Practice Run:**
Every 6 months, pretend you've been asked to submit a complete document package. Time yourself assembling it. If it takes more than 30 minutes, you need to improve your system.
